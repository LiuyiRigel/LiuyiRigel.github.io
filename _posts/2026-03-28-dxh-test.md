# 二维注热开采天然气水合物 COMSOL 模型（新手可跑通版）

## 📌 模型目标

构建一个**二维注热开采天然气水合物的基础数值模型**，实现以下物理过程：

* 多孔介质渗流（Darcy）
* 热传导 + 对流
* 水合物分解（简化动力学）
* 注热驱动分解过程

该模型特点：

* ✔ 可直接运行
* ✔ 耦合关系清晰
* ✔ 易于扩展（适合后续论文升级）

---

## 🧩 1. 创建模型

### 操作步骤

* 打开 Model Wizard
* 选择 **2D**
* 添加物理场：

  * Darcy’s Law（dl）
  * Heat Transfer in Porous Media（ht）
  * Domain ODEs and DAEs（ode）
* Study 类型选择：

  * Time Dependent

---

## 📐 2. 几何（Geometry）

### 操作

* 添加 Rectangle

### 参数

```text
宽度：10 m
高度：5 m
```

## ⚙️ 3. 参数（Definitions → Parameters）

```comsol
phi0 = 0.4
k0 = 1e-13 [m^2]

rho = 1000 [kg/m^3]
Cp = 4200 [J/(kg*K)]
lambda = 2 [W/(m*K)]

T_inj = 350 [K]
T0 = 280 [K]

p_out = 3e6 [Pa]
p0 = 5e6 [Pa]

k_react = 1e-5
peq = 4e6

C1 = -5e4
C2 = 0
Mh = 0.016

h = 100
```

---

## 🔧 4. 变量（Definitions → Variables）

```comsol
Ceq = phi0*rho*Cp + (1-phi0)*2600*800
keq = lambda

Qh = (Sh/Mh)*(C1 + C2*T)
q = h*(T_inj - T)

phi = phi0*(1 - Sh)
k = k0*(phi/phi0)^3
```

---

## 🌊 5. Darcy’s Law（dl）

### Domain 设置

```comsol
Permeability: k
Porosity: phi
Density: rho
Viscosity: 1e-3
```

---

### 边界条件

* 左边界：No Flow
* 右边界：

```comsol
p = p_out
```

* 其他边界：No Flow

---

## 🌡️ 6. Heat Transfer（ht）

### Domain 设置

```comsol
Heat capacity: Ceq
Thermal conductivity: keq
```

---

### 勾选物理机制

```text
✔ Conduction
✔ Convection
```

---

### 速度场（关键耦合）

```comsol
[dl.u, dl.v]
```

---

### 热源项

```comsol
Qh + q
```

---

### 边界条件

* 左边界（注热）：

```comsol
T = T_inj
```

* 其他边界：


## ⚗️ 7. 水合物分解（Domain ODE）

### 添加变量

```text

```

---

### 控制方程

```comsol
d(Sh,t) = -k_react*max(p - peq, 0)
```

# COMSOL 多孔介质传热中“体积热容”设置（含参数示例）

## 📌 结论

```text
无需单独输入“体积热容”，
通过 Matrix + Fluid 自动形成等效热容
```

---

## ⚙️ 参数（Definitions → Parameters）

```comsol
rho_w = 1000      [kg/m^3]
rho_g = 50        [kg/m^3]
rho_h = 900       [kg/m^3]

Cw = 4200         [J/(kg*K)]
Cg = 2000         [J/(kg*K)]
Ch = 2100         [J/(kg*K)]

rho_s = 2600      [kg/m^3]
Cs = 800          [J/(kg*K)]
lambda_s = 2.5    [W/(m*K)]

phi = 0.4
Sw = 0.3
```

---

## 🔧 变量（Definitions → Variables）

```comsol
Sh = 0.4
Sg = 1 - Sw - Sh
```

---

## 🌊 Fluid（流体）设置

### Density

```comsol
rho_f = rho_w*Sw + rho_g*Sg + rho_h*Sh
```

### Heat Capacity

```comsol
Cp_f = (rho_w*Sw*Cw + rho_g*Sg*Cg + rho_h*Sh*Ch)/(rho_w*Sw + rho_g*Sg + rho_h*Sh)
```

---

## 🪨 Porous Matrix（固体骨架）

```comsol
Density: rho_s
Heat capacity: Cs
Thermal conductivity: lambda_s
```

## 📦 变量说明

| 变量       | 含义     | 数值   | 单位       | 位置         |
| -------- | ------ | ---- | -------- | ---------- |
| rho_w    | 水密度    | 1000 | kg/m³    | Parameters |
| rho_g    | 气密度    | 50   | kg/m³    | Parameters |
| rho_h    | 水合物密度  | 900  | kg/m³    | Parameters |
| Cw       | 水比热    | 4200 | J/(kg·K) | Parameters |
| Cg       | 气比热    | 2000 | J/(kg·K) | Parameters |
| Ch       | 水合物比热  | 2100 | J/(kg·K) | Parameters |
| rho_s    | 岩石密度   | 2600 | kg/m³    | Parameters |
| Cs       | 岩石比热   | 800  | J/(kg·K) | Parameters |
| lambda_s | 导热率    | 2.5  | W/(m·K)  | Parameters |
| phi      | 孔隙率    | 0.4  | -        | Parameters |
| Sw       | 水饱和度   | 0.3  | -        | Parameters |

| Sg       | 气饱和度   | 0.3  | -        | Variables  |

## 🧠 核心逻辑

[
(ρC_p)_{eff} = (1-φ)ρ_s C_s + φ ρ_f C_f
]

```text
Fluid → 定义混合物性质
Matrix → 定义骨架性质
COMSOL → 自动体积平均
```
