# 二维注热开采天然气水合物 COMSOL 模型（新手可跑通版）

## 📌 模型目标

构建一个**二维注热开采天然气水合物的基础数值模型**，实现以下物理过程：

* 多孔介质渗流（Darcy）
* 热传导 + 对流
* 水合物分解（简化动力学）
* 注热驱动分解过程

该模型特点：

* ✔ 可直接运行
* ✔ 耦合关系清晰
* ✔ 易于扩展（适合后续论文升级）

---

## 🧩 1. 创建模型

### 操作步骤

* 打开 Model Wizard
* 选择 **2D**
* 添加物理场：

  * Darcy’s Law（dl）
  * Heat Transfer in Porous Media（ht）
  * Domain ODEs and DAEs（ode）
* Study 类型选择：

  * Time Dependent

---

## 📐 2. 几何（Geometry）

### 操作

* 添加 Rectangle

### 参数

```text
宽度：10 m
高度：5 m
```

---

## ⚙️ 3. 参数（Definitions → Parameters）

```comsol
phi0 = 0.4
k0 = 1e-13 [m^2]

rho = 1000 [kg/m^3]
Cp = 4200 [J/(kg*K)]
lambda = 2 [W/(m*K)]

T_inj = 350 [K]
T0 = 280 [K]

p_out = 3e6 [Pa]
p0 = 5e6 [Pa]

k_react = 1e-5
peq = 4e6

C1 = -5e4
C2 = 0
Mh = 0.016

h = 100
```

---

## 🔧 4. 变量（Definitions → Variables）

```comsol
Ceq = phi0*rho*Cp + (1-phi0)*2600*800
keq = lambda

Qh = (Sh/Mh)*(C1 + C2*T)
q = h*(T_inj - T)

phi = phi0*(1 - Sh)
k = k0*(phi/phi0)^3
```

---

## 🌊 5. Darcy’s Law（dl）

### Domain 设置

```comsol
Permeability: k
Porosity: phi
Density: rho
Viscosity: 1e-3
```

---

### 边界条件

* 左边界：No Flow
* 右边界：

```comsol
p = p_out
```

* 其他边界：No Flow

---

## 🌡️ 6. Heat Transfer（ht）

### Domain 设置

```comsol
Heat capacity: Ceq
Thermal conductivity: keq
```

---

### 勾选物理机制

```text
✔ Conduction
✔ Convection
```

---

### 速度场（关键耦合）

```comsol
[dl.u, dl.v]
```

---

### 热源项

```comsol
Qh + q
```

---

### 边界条件

* 左边界（注热）：

```comsol
T = T_inj
```

* 其他边界：

```text
Thermal Insulation
```

---

## ⚗️ 7. 水合物分解（Domain ODE）

### 添加变量

```text
变量名：Sh
初始值：0.4
```

---

### 控制方程

```comsol
d(Sh,t) = -k_react*max(p - peq, 0)
```

---

## 🧊 8. 初始条件

```comsol
T = T0
p = p0
Sh = 0.4
```

---

## 🔲 9. 网格（Mesh）

Physics-controlled mesh

## 📊 11. 结果分析

### 预期现象

#### 温度场

* 左侧高温向右扩散

#### 水合物饱和度（Sh）

* 从左侧开始降低（分解）

#### 压力场

* 从左向右驱动流动

---

## ⚠️ 12. 常见问题与解决

### 问题1：模型发散

```comsol
k_react = 1e-6
```

---

### 问题2：没有对流

检查：

```text
✔ 是否勾选 Convection
✔ 是否填写 velocity field
```

---

### 问题3：Sh 变为负值

```comsol
d(Sh,t) = -k_react*max(p - peq, 0)
```

## 📌 使用建议

建议按以下顺序逐步提升模型：

```text
单相流 → 注热 → 分解 → 孔隙率 → 渗透率 → THM耦合
```

避免一开始就做复杂耦合导致模型不收敛。
